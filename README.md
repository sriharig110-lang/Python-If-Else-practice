# Python If-Else Practice

This repository contains 10 beginner Python practice programs created while learning Python fundamentals.

## Topics Covered

- if
- if...else
- if...elif...else
- Comparison Operators
- Logical Operators
- Nested if
- Input Validation

## Programs

1. Even or Odd
2. Positive, Negative or Zero
3. Largest of Two Numbers
4. Largest of Three Numbers
5. Grade Calculator
6. Leap Year Checker
7. Simple Calculator
8. Vowel or Consonant Checker
9. Even or Odd (User Input)
10. Password Strength Checker

## Language

Python
